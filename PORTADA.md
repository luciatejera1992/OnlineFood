# OnlineFood

Proyecto: Análisis Exploratorio de Datos — OnlineFood

Autor: Lucía Derulos

Fecha: 12 de enero de 2026

---

Resumen

OnlineFood explora un conjunto de datos de pedidos de comida online para entender patrones de consumo, comportamiento por hora/día, distribución por establecimientos, y métricas clave como tiempo de entrega o valor medio del pedido.

Objetivo de la portada

Presentar de forma clara y visual el propósito del proyecto y facilitar a un lector nuevo la navegación hacia los artefactos principales (notebook y dataset).

Contenido clave

- Notebook de EDA: `EDA/EDA.ipynb`
- Dataset: `EDA/onlinefoods.csv` (si está incluido)
- Dependencias: `requirements.txt`

Cómo usar esta portada

1. Lee el resumen para entender el foco del análisis.
2. Abre el notebook `EDA/EDA.ipynb` para ver el análisis paso a paso.
3. Reproduce el entorno con `requirements.txt` y ejecuta el notebook localmente.
